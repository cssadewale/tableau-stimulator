# Tableau Stimulator Enterprise — Quick Start

## 10 Steps to Your First Enterprise Analysis

1. Open `index.html` in a browser, or deploy to GitHub Pages / Netlify
2. The **Enterprise splash screen** appears — wait 2 seconds for it to dismiss
3. Click **Sample Data** in the menu strip to load the built-in sales dataset, or upload your own CSV/TSV
4. Drag a dimension (e.g., Region) to **COLS** and a measure (e.g., Sales) to **ROWS**
5. Click **Render** or use **Show Me** for a Tableau-style chart suggestion
6. Try an **Enterprise chart**: select 🌳 Treemap, 📊 Histogram, or 🌹 Rose from the dropdown
7. Click **Explain Data** in the menu strip for algorithmic insights (no AI API)
8. Click **Regression** or **Clustering** in the menu strip for advanced analytics
9. Build dashboards (4-column layout available), stories, pivot tables, and reports
10. Use the **Export Center** for PNG, SVG, CSV, HTML, JSON, PDF, and more

## Enterprise-Specific Features
- **Explain Data** — 8 algorithmic insights per chart
- **Linear Regression** — R², slope, intercept, standard error
- **K-Means Clustering** — Adds _Cluster field to data
- **Data Stories** — Capture charts as story points, export as HTML presentation
- **RLS** — Row-Level Security simulation
- **8 new chart types** — Treemap, Sunburst, Sankey, Parallel, Bubble, Rose, Histogram, Bullet
- **Export Center** — 9 formats in one place

No AI API, backend, database, or paid service required.
