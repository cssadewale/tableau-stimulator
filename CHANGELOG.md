# Changelog — Tableau Stimulator Enterprise

## [v9.0.0] Enterprise Edition — 2026-05-25

### Added — Enterprise Features
- **Treemap Chart** — Hierarchical area visualization with proportional rectangles
- **Sunburst Chart** — Multi-level radial hierarchy with Color By drill-down
- **Sankey Flow Diagram** — Source-to-target flow visualization with gradient links
- **Parallel Coordinates** — Multi-dimensional comparison across all numeric fields
- **Bubble Chart** — Enhanced scatter with Size By encoding for 3-variable analysis
- **Nightingale Rose** — Polar area chart for cyclical magnitude comparison
- **Histogram** — Auto-binned (15 bins) frequency distribution
- **Bullet Chart** — Actual vs target with qualitative range bands
- **Data Stories / Storyboard** — Capture charts as story points with narrative captions, navigation, and standalone HTML export
- **Explain Data** — Algorithmic analysis engine generating 8 structured insights (no AI API)
- **Linear Regression** — Full OLS: R², slope, intercept, standard error, Pearson r, equation
- **K-Means Clustering** — Unsupervised grouping (K=2-8), adds _Cluster field to data
- **Enterprise Export Center** — 9 export formats (PNG, SVG, CSV, HTML, JSON, PDF, Pivot, Stats, Dictionary)
- **Row-Level Security Simulation** — Restrict rows by field=value, toggle on/off
- **4-Column Dashboard Layout** — Dense enterprise grid option
- **SVG Vector Export** — Scalable vector graphics for print quality
- **Enterprise Splash Screen** — Branded loading screen with creator details
- **Enterprise Status Bar** — Version badge, HMG links, portfolio link
- **Enterprise Footer** — Creator attribution bar with links
- **Enterprise Features Modal** — Detailed guide for all 16 new features
- Removed Cloudflare injection scripts from deployment artifacts
- Fixed email addresses previously obscured by Cloudflare protection

### Preserved — All Original Features
- All 75+ original features from Tableau Stimulator v8 are preserved
- All 15 original chart types remain fully functional
- Worksheet, Pivot, Statistics, Data Cleaner, Compare, Report, Dashboard tabs unchanged
- Show Me, Ask Data, Smart Narrative, Formula Bar, Undo/Redo unchanged
- All keyboard shortcuts, themes, palettes, and export functions unchanged
