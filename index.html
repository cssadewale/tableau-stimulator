👤 About the Creator & Platform
Tableau Stimulator is an independent, free, Tableau-style analytics simulator built with vanilla HTML, CSS and JavaScript. It recreates familiar BI workflows — shelves, marks, worksheets, dashboards, stories/reports, Show Me-style templates and analytics overlays — without paid APIs or proprietary Tableau assets.

⚡ Enterprise Features Guide
Tableau Stimulator Enterprise adds professional-grade features on top of the full platform. All free, local, no AI API.
🌳
Treemap Chart
Hierarchical area visualization. Drag dimension to COLS and measure to ROWS. Each rectangle size = aggregated value. Best for composition of whole.
☀
Sunburst Chart
Multi-level radial hierarchy. COLS + ROWS + Color By for 2-level drill. Inner = parent, outer = children.
🔀
Sankey Flow
Flow visualization. COLS=Source, Color By=Target, ROWS=Value. Line thickness = volume. Customer journeys, budget flows.
═
Parallel Coordinates
Multi-dimensional comparison. Every numeric field = axis, every row = line. Reveals patterns across dimensions.
⚪
Bubble Chart
Scatter + size encoding. COLS=X, ROWS=Y, Size By=bubble diameter. Three-variable comparison.
🌹
Nightingale Rose
Polar area chart. Sector radius = magnitude. Better than pie for absolute value comparison on cyclical data.
📊
Histogram
Auto-binned frequency distribution. Drag numeric to ROWS. 15 bins showing count. Data shape + outlier detection.
▰
Bullet Chart
Compact KPI: actual vs target + quality bands. COLS + ROWS + Goal Line. Replaces gauges in dense dashboards.
📖
Data Stories
Capture charts as story points with captions. Navigate + present. Export as standalone HTML. Tableau Story replica.
🔍
Explain Data
Algorithmic insights: dominant category, distribution, variability, trend, concentration, recommendations. No AI API.
📈
Linear Regression
Full OLS: R², slope, intercept, standard error. Select X + Y fields. Regression equation + interpretation.
🎯
K-Means Clustering
Unsupervised grouping. Select 2 fields, choose K. Results: centroids, sizes, _Cluster field added to data.
💾
Export Center
9 formats: PNG, SVG, CSV, HTML, JSON, PDF, pivot, stats, dictionary. All local processing.
🔐
Row-Level Security
Simulate RLS: restrict visible rows by field value. Toggle on/off. Great for training and governance demos.
📊
Enhanced Dashboard
4-column layouts, cross-filtering, KPI cards, presentation mode. Enterprise-grade dashboard builder.
